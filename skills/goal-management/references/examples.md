## 使用例

### 例1: 年間目標の設定

**ユーザー**: 「2025年の年間目標を立てたい」

→ Claude:
1. プロジェクトルート/goals/2025-年間目標.md の存在をチェック
2. my-profile/ を読み込み（プロフィール情報を確認）
3. 年間テーマを提案（例: 「フルスタックエンジニアとして自走できるようになる」）
4. Value別の年間目標を提案
5. 成果指標（KPI）を設定
6. 年間目標を2つの半期目標（H1・H2）に分解
7. goals/2025-年間目標.md に保存

### 例2: 半期目標の設定（年間目標あり）

**ユーザー**: 「今期（2025H2）の目標を立てたい」

→ Claude:
1. goals/2025-年間目標.md を読み込み（年間目標を確認）
2. my-profile/ を読み込み
3. 前回評価のB/C項目を自動抽出
4. AskUserQuestionで課題を深掘り
   - 「現在、最も改善したいと感じている課題は何ですか？」
   - 「なぜその課題が発生していると思いますか？」（5 Whysで深掘り）
   - 根本原因を特定
   - 改善策を検討
5. 年間目標を踏まえて半期目標を提案
6. 課題改善を半期目標に組み込む
7. 半期→月次→週次に分解
8. goals/2025-H2-半期目標.md、2025-H2-月次目標.md、2025-H2-週次目標.md に保存

### 例3: 半期目標の設定（年間目標なし）

**ユーザー**: 「今期の目標を立てたい」

→ Claude: my-profile/ をチェック
  - **my-profile/ がある場合**:
    - my-profile/ を読み込み（現在のグレード、目標グレード、前回評価などを確認）
    - 前回評価のB/C項目を自動抽出
    - AskUserQuestionで課題を深掘り
    - 現在のグレード要件を確認し、目標グレードを目指せる具体的な目標を提案
    - **各目標に評価項目との対応を明記**
    - 課題改善を半期目標に組み込む
    - 半期→月次→週次に分解

  - **my-profile/ がない場合**:
    - 「目標設定の前に、まずプロフィールを作成しましょう。
       profile-builder skill を使って、30分-1時間かけて高解像度のプロフィールを作成してください。
       プロフィールは目標設定の土台となる重要なステップです。」

### 例4: 課題の深掘りのみ

**ユーザー**: 「最近うまくいっていないことがあるので、整理したい」

→ Claude:
1. 「今、どんな課題や悩みを感じていますか？自由に話してください。」（壁打ち形式）
2. 話の中から課題のキーワードを抽出
3. AskUserQuestionで各課題を深掘り（5 Whys）
4. 根本原因を特定
5. 改善策を検討
6. 「半期目標に組み込みますか？」と確認

**ユーザー**: 「目標の達成状況を確認したい」
→ maclogger-review スキルを使用してください（このスキルは目標設定専門です）
