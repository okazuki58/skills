# Skills

開発ワークフローと学習を効率化するClaude Skills集

## インストール

マーケットプレイスを追加してからインストール：

```bash
/plugin marketplace add okazuki58/skills
/plugin install okazuki-skills@okazuki-skills
```

または対話的なUIで選択：

```bash
/plugin
```

## 利用可能なSkills

### goal-management
年間目標→半期→月次→週次の階層的な目標分解、課題の深掘り、評価基準に基づいた目標設定をサポート。

### maclogger-review
週次・月次・半期の振り返り作成。目標達成状況の確認と次のアクションを明確化。

### profile-builder
10分で完成する簡潔なエンジニアプロフィール作成。経験年数・技術理解度からレベル別の課題を自動生成。

### learning-support
「〇〇を勉強したい」と言うだけで、学習計画立案→GitHub Projects/Issue作成→ハンズオン学習サポートまで自動化。

### task-management
リポジトリベースのタスク管理支援。対話的計画立案→タスク分解→GitHub Projects/Issue作成まで自動化。

### spec-to-code
仕様駆動・テスト駆動開発ワークフロー。仕様インタビュー→ドキュメント化→GitHub Issue生成→TDD実践を支援。
