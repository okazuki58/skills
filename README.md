# Skills

開発ワークフローを効率化するClaude Skills集

## 🚀 インストール

```bash
/plugin install https://github.com/okazuki58/skills
```

## 📁 利用可能なSkills

### spec-to-code

仕様策定から実装までの開発ワークフローをガイドします。

**使用シーン:**
- 新機能の開発開始時
- 要件が曖昧で明確化が必要な時
- テスト駆動開発（TDD）を実践したい時
- 構造化されたGitHub Issueを作成したい時

**使用例:**
```
@spec_template.md を読んで、カート機能についてインタビューしてください。
```

## 🌟 特徴

- **言語・フレームワーク非依存**: どんな技術スタックでも使える
- **TDD重視**: テスト駆動開発のベストプラクティスを実践
- **仕様の明確化**: 構造化された要件定義
